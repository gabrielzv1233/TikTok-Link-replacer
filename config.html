Variable Guide

Use the following variables in your replacement text or link template:

  • {link}: The full original TikTok link.
  • {query}: The query string (e.g., ?lang=en).
  • {creator}: The creator's username (e.g., gabrielzv1233, If link is to a tag, it may get the tag instead).
  • {video}: The video ID (e.g., 7041397012427836678).
  • {content}: The original text of the <a> element.

Error Handling:

  • err::500: A critical error occurred/element not found.
